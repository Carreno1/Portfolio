FIZZ_BUZZ Challenge

♦

Fizz Buzz App will take numbers inputed and print numbers 1 through 100. "Fizz" at perfect divisibles of the first input, "Buzz" at perfect divisibles of the second input, and "FizzBuzz" when the number satisfies both conditions. 


                                
        $("#btnFizzBuzz").on("click", function () {

            var fizznum = $("#fizznum").val();

            var buzznum = $("#buzznum").val();

            
            var arrOutput = new Array();

            if (fizznum.length < 1 || buzznum.length < 1) {
                alert("No input");
                return;
            }
            var meme = 2;
            fizznum = Number(fizznum);
            buzznum = Number(buzznum);

            for (i = 1; i <= 100; i++) {

                if (i % fizznum === 0 && i % buzznum === 0) {
                    console.log("FizzBuzz");
                    arrOutput.push("FIZZ-BUZZ");
                } else if (i % fizznum === 0) {
                    arrOutput.push("FIZZ");
                } else if (i % buzznum === 0) {
                    arrOutput.push("BUZZ");
                } else {
                    arrOutput.push(i);
                }


            }


            $("#fizzBuzzOutput").html(arrOutput.join(", "));
        });

TacoCat Challenge

♦

The TacoCat Challenge asks us to take an inputed string and reverse it. Along with this the output will indicate whether or not a palindrome was inputed.


                       
        $("#btnTacoCat").on("click", function () {

            var word = String($("#txtTacoCat").val()).toLowerCase();

            var isPalin = "";
            var revWord = "";

            for (var loop = word.length - 1; loop >= 0; loop--) {
                revWord += word[loop];
            }

            if (word === revWord) {
                isPalin += "a Palindrome";
            } else {
                isPalin += "not a Palindrome";
            }

            $("#tcOutput").html(`The reverse of ${word} is: ${revWord}. This item is ${isPalin}`);

        });

Sunset Hills

♦

This challenge asks us to be able to take in varying heights of buildings as they are located from west to east. Then it shows the user how many buildings can successfully see over the rest and as a result, the sun set.



        $("#btnSunsetHills").on("click", function () {

            let arr = Array();

            let sunset_arr = Array();

            sunset_arr.push(Number($("#sunnum1").val()));

            arr.push(Number($("#sunnum1").val()));
            arr.push(Number($("#sunnum2").val()));
            arr.push(Number($("#sunnum3").val()));
            arr.push(Number($("#sunnum4").val()));
            arr.push(Number($("#sunnum5").val()));
            arr.push(Number($("#sunnum6").val()));

            var height_in_front = arr[0];

            for (i = 1; i < arr.length; i++) {
                if (arr[i] > height_in_front) {
                    sunset_arr.push(arr[i]);
                    if (arr[i] > height_in_front) {
                        height_in_front = arr[i];
                    }
                }
            }

            $("#sunOutput").html(`${sunset_arr.length} --> ${sunset_arr}`);

        });

Sum of All Fears

♦

This challenge asks us to take in a number, K, and then an array of other values. The output tells us whether or not any two values present can add up to K.



        $("#btnSoAF").on("click", function () {

            var k = Number($("#k").val());
            let arr = new Array();
            var addup = false;
            let answer = "";
            arr.push(Number($("#sumnum1").val()));
            arr.push(Number($("#sumnum2").val()));
            arr.push(Number($("#sumnum3").val()));
            arr.push(Number($("#sumnum4").val()));

            for (i = 0; i < arr.length; i++) {
                for (j = 1; j < arr.length; j++) {
                    if (arr[i] + arr[i + j] === k) {
                        addup = true;
                    }
                }
            }

            if (addup) {
                answer += "yes";
            } else {
                answer += "no";
            }

            $("#soAFoutput").html(answer);

        });
